## Stable Diffusion 1.5 Prompting Style

**Format:** Tag-based with weighted emphasis. Mix of natural language and tags.

**Positive prompt structure:**
```
[subject], [details], [environment], [lighting], [quality tags], [style]
```

**Quality tags:**
```
masterpiece, best quality, highly detailed, sharp focus, professional
```
For photos: `RAW photo, DSLR, 50mm, f/1.8, photorealistic`
For anime: `anime, cel shading, vibrant colors`

**Weight syntax:** `(important word:1.3)` — range 0.5 to 1.5 typical.

**Negative prompt (very important for SD1.5):**
```
(worst quality:1.4), (low quality:1.4), normal quality, lowres, bad anatomy, bad hands, extra fingers, missing fingers, extra limbs, deformed, ugly, duplicate, morbid, mutilated, poorly drawn hands, poorly drawn face, mutation, blurry, watermark, text, signature
```

**Textual inversions / embeddings:**
SD1.5 has many popular embeddings:
- `EasyNegative` — replaces long negative prompts
- `bad_prompt_version2` — quality improvement
- `ng_deepnegative_v1_75t` — deep negative

**Key settings:**
- Resolution: 512x512, 512x768, 768x512 (NEVER above 768 without hires fix)
- CFG: 7-11
- Sampler: `dpmpp_2m` or `euler_a` with `karras`
- Steps: 20-30
- Hires fix recommended for anything above 512px (denoise 0.4-0.6)

**Do NOT:**
- Generate at resolutions above 768px directly (causes duplicates/artifacts)
- Use score tags (those are for Pony)
- Use very long natural language (tags work better for SD1.5)
- Forget negative prompts (critical for SD1.5 quality)
